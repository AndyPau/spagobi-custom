JOB

Description

Root tag

Parameters

Attribute Description Required
jobName The name of the job to execute Yes
version The job version. If not specified the engine will assume 0.1 as default version number. If the version number does not match with the one of the referenced deployed job the engine wont be able to locate it in the runtime repository. Yes
project The name of the talend project in which your job has been created Yes
language The esportation language used (Java or Perl) Yes
context The name of the target context to use with the job. If not specified DEFAULT will be used as default value No

Nested elements

noone

Examples

<ETL version='2.0.0'>
  <JOB> 
    project="DEMO_PROJECT"
    jobName="myFirstJob"
    version="2.4"
    language="java" 
    context="Default"     
  <JOB/> 
</ETL>
