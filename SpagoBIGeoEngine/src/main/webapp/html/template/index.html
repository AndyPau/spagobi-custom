MAP

Description

Root tag

Parameters

Attribute Description Required
version Version of the engine required to execute this template No

Nested elements

MAP_PROVIDER

The nested MAP_PROVIDER element specifies the IMapProvider implemetation to use and its configuration block. The MapProvider is the SpagoBIGeoEngine's component that have the responsability to provide maps used for the analysis at hand.

DATAMART_PROVIDER

The nested DATAMART_PROVIDER element specifies the IDataMartProvider implementation to use and its configuration block. The MapProvider is the SpagoBIGeoEngine's component that have the responsability to provide the datamart used for the analysis at hand.

MAP_RENDERER

The nested MAP_RENDERER element specifies the IMapRenderer implementation to use and its configuration block. The MapRenderer is the SpagoBIGeoEngine's component that have the responsability to use the map retrived by the MapProvidere and the datamart retrived by the DataMartProvider in order to produce the final analysis result

Examples

<MAP version='2.0.0'>
  <MAP_PROVIDER/>
  <DATAMART_PROVIDER/>
  <MAP_RENDERER/>
</MAP>
