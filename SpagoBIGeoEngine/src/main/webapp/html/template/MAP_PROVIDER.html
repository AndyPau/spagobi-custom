MAP_PROVIDER

Description

MAP_PROVIDER element specifies the IMapProvider implemetation to use and its configuration block. The MapProvider is the SpagoBIGeoEngine's component that have the responsability to provide maps used for the analysis at hand.

Parameters

Attribute Description Required
class_name The implementation of the interface IMapProvider to use in order to retrive maps. If not specified the class SOMapProvider is used as default implementation. This class retrive maps directly from the SpagoBI's map catalogues by means of a dedicated web service. No
map_name The name of the map to use as analisys entry point Yes

Nested elements

none

Examples

<MAP_PROVIDER 
    class_name="it.eng.spagobi.engines.geo.map.provider.SOMapProvider" 
    map_name="USA_REGIONS" />
