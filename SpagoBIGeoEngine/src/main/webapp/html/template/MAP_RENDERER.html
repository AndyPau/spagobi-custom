MAP_RENDERER

Fully qualified name:    MAP.MAP_RENDERER Back - Up - Home

Description

MAP_RENDERER element specifies the IMapRenderer implementation to use and its configuration block. The MapRenderer is the SpagoBIGeoEngine's component that have the responsability to use the map retrived by the MapProvidere and the datamart retrived by the DataMartProvider in order to produce the final analysis result

Parameters

Attribute Description Required
class_name The implementation of the interface IMapRenderer to use in order to retrive maps. If not specified the class InteractiveMapRenderer is used as default implementation. No

Nested elements

MEASURES

The nested MEASURES element contains a complete list of all measures used in the analysis

LAYERS

The nested LAYERS element specifies ...

GUI_SETTINGS

The nested GUI_SETTINGS element specifies ...

Examples


<MAP_RENDERER 
  class_name="it.eng.spagobi.engines.geo.map.renderer.InteractiveMapRenderer">
  
  <MEASURES/>
  <LAYERS/>
  <GUI_SETTINGS/>

</MAP_RENDERER>
